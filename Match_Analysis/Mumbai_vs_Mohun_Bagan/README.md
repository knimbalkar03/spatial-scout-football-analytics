# Mumbai City vs Mohun Bagan — ISL Tactical Analysis

## Project Overview

This project analyzes the ISL match between Mumbai City FC and Mohun Bagan using event data and spatial football analytics.

The objective was to identify how Mumbai City controlled possession, progression, and territorial dominance throughout the match.

## Tools Used

- Python
- Pandas
- mplsoccer
- Matplotlib
- StatsBomb-style event analysis

## Key Findings

- Mumbai City controlled central progression through structured passing sequences.
- The pass network highlights strong connectivity between midfield and attacking units.
- Pressing actions were concentrated in advanced areas, allowing Mumbai to regain possession quickly.
